## Cross-Phase Integration

### Wired (14/16)
All core integration paths verified: channel protocol → worker → AgentHandle → transports → CompanyRoot → Discord routing → health tree → reconnect.

### Degraded (2/16)

**1. dispatch() socket bypass (HEAD-01)**
`RuntimeAPI.dispatch()` spawns workers in stdio mode without `--socket` flag. Stale socket references on AgentHandle make respawned workers unreachable. Fix: use CompanyRoot.hire() spawn path.

**2. Health polling missing (HEAD-03)**
No background task sends `HealthCheckMessage`. Health data goes stale after 120s. Workers handle it correctly but it's never triggered. Fix: add periodic polling task in CompanyRoot.
