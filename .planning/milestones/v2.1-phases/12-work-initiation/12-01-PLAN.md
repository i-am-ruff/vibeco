**container.py:** Three changes:

1. Add `import logging` and `logger = logging.getLogger(__name__)` at module level (after existing imports).

2. Add `_wait_for_claude_ready` async method to AgentContainer:
```python
async def _wait_for_claude_ready(
    self,
    pane,
    timeout: float = 60.0,
    poll_interval: float = 1.0,
) -> bool:
```
- Uses `time.monotonic()` deadline loop
- Calls `await asyncio.to_thread(self._tmux.get_output, pane)` each iteration
- Wraps get_output in try/except (return empty list on exception, continue polling)
- Scans lines in reverse for last non-empty stripped line
- Ready when `stripped == ">"` or `stripped.endswith(" >")`
- Sleeps `poll_interval` between checks via `await asyncio.sleep(poll_interval)`
- Returns True when ready, False on timeout
- Add `import time` at top of file

3. Rewrite `_launch_tmux_session`:
- Keep existing: create session, create pane, set pane_id, build and send launch command
- Replace `await asyncio.sleep(3)` with `await asyncio.sleep(2)` (trust dialog wait)
- Keep the `pane.send_keys("", enter=True)` for trust dialog acceptance
- After trust acceptance, check `if self.context.gsd_command:` -- if so, call `_wait_for_claude_ready(pane)`. If ready, `await asyncio.to_thread(self._tmux.send_command, pane, self.context.gsd_command)` and log info. If not ready, log warning. If no gsd_command configured, do nothing extra (backward compatible).
