```python
class BacklogItemStatus(str, Enum):
    PENDING = "pending"
    ASSIGNED = "assigned"
    IN_PROGRESS = "in_progress"
    COMPLETED = "completed"
    CANCELLED = "cancelled"

class BacklogItem(BaseModel):
    item_id: str
    title: str
    description: str = ""
    priority: int = 0
    status: BacklogItemStatus = BacklogItemStatus.PENDING
    assigned_to: str | None = None
    created_at: str

class BacklogQueue:
    def __init__(self, memory: MemoryStore) -> None: ...
    async def load(self) -> None: ...
    async def append(self, item: BacklogItem) -> None: ...
    async def insert_urgent(self, item: BacklogItem) -> None: ...
    async def insert_after(self, after_id: str, item: BacklogItem) -> None: ...
    async def reorder(self, item_id: str, new_position: int) -> None: ...
    async def cancel(self, item_id: str) -> None: ...
    async def claim_next(self, agent_id: str) -> BacklogItem | None: ...
    async def mark_completed(self, item_id: str) -> None: ...
    async def mark_pending(self, item_id: str) -> None: ...
    @property
    def pending_items(self) -> list[BacklogItem]: ...
```

    1. Create `src/vcompany/autonomy/project_state.py` with `ProjectStateManager`:
       - __init__(self, backlog: BacklogQueue, memory: MemoryStore) -- takes PM's backlog and PM's memory store
       - async assign_next_task(self, agent_id: str) -> BacklogItem | None:
         Calls backlog.claim_next(agent_id). If item claimed, stores assignment as JSON in PM's MemoryStore under key f"assignment:{agent_id}". Returns the item.
       - async handle_task_completed(self, agent_id: str, item_id: str) -> None:
         Calls backlog.mark_completed(item_id). Deletes PM's MemoryStore key f"assignment:{agent_id}".
       - async handle_task_failed(self, agent_id: str, item_id: str) -> None:
         Calls backlog.mark_pending(item_id). Deletes PM's MemoryStore key f"assignment:{agent_id}".
       - async reassign_stale(self, active_agent_ids: set[str]) -> list[str]:
         Iterates backlog items with status ASSIGNED. If assigned_to not in active_agent_ids, calls mark_pending. Returns list of reassigned item_ids.
       - async get_agent_assignment(self, agent_id: str) -> dict | None:
         Reads PM's MemoryStore key f"assignment:{agent_id}", returns parsed JSON or None.

    2. Extend `src/vcompany/agent/fulltime_agent.py`:
       - Add optional `backlog: BacklogQueue | None = None` attribute (set after start via property or direct assignment, not constructor -- keep constructor signature stable).
       - Add `_project_state: ProjectStateManager | None = None` attribute.
       - Override start(): after super().start(), if backlog is set, load it and create ProjectStateManager.
       - Override _handle_event to route events:
         "task_completed" -> _project_state.handle_task_completed(event["agent_id"], event["item_id"])
         "task_failed" -> _project_state.handle_task_failed(event["agent_id"], event["item_id"])
         "add_backlog_item" -> backlog.append(BacklogItem(**event["item"]))
         "request_assignment" -> _project_state.assign_next_task(event["agent_id"])
         Unknown types: log warning, no-op (do not raise).

    3. Extend `src/vcompany/agent/gsd_agent.py`:
       - Add `async def get_assignment(self) -> dict | None`:
         Reads "current_assignment" from self.memory (own MemoryStore). Returns parsed JSON or None.
       - Add `async def set_assignment(self, assignment: dict) -> None`:
         Writes assignment as JSON to "current_assignment" in self.memory.
       - Add `def make_completion_event(self, item_id: str, result: str = "success") -> dict`:
         Returns {"type": "task_completed", "agent_id": self.context.agent_id, "item_id": item_id, "result": result}.
       - Add `def make_failure_event(self, item_id: str, reason: str = "") -> dict`:
         Returns {"type": "task_failed", "agent_id": self.context.agent_id, "item_id": item_id, "reason": reason}.

    - Create BacklogQueue with PM's MemoryStore.
    - Test full flow: add item -> assign -> complete -> verify COMPLETED status.
    - Test failure flow: add item -> assign -> fail -> verify back to PENDING.
    - Test crash safety: add item -> assign -> (simulate crash by not calling handle_task_completed) -> reassign_stale recovers it.
    - Test FulltimeAgent event routing with real backlog.
    - Test GsdAgent assignment read/write.

    IMPORTANT: The crash safety guarantee comes from the single-writer pattern -- only PM writes to the backlog. Agents never write to PM's MemoryStore. They post events to PM's queue. If agent crashes, the event was either posted or it was not. PM's state remains consistent.
