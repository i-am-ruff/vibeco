Create `src/vcompany/bot/cogs/commands.py` implementing CommandsCog with module-level `async def setup(bot)` function per discord.py Cog pattern.

**Class: CommandsCog(commands.Cog)**

Constructor takes `bot: VcoBot`, stores as `self.bot`.

**All commands decorated with `@is_owner()` per D-09/DISC-10.**

**All AgentManager/file calls wrapped in `asyncio.to_thread()` per D-14/DISC-11.**

Commands:

1. **`!new-project [name]`** per D-01, D-02, DISC-03:
   - If bot not ready (self.bot.is_bot_ready is False), reply "Bot is starting up, please wait..."
   - If no name argument, reply asking for a project name
   - Get guild, get owner_role (discord.utils.get by name "vco-owner")
   - Call `await setup_project_channels(guild, name, owner_role, self.bot.project_config.agents)`
   - Create a thread in the current channel for project discussion: `await ctx.channel.create_thread(name=f"Project: {name}", message=ctx.message)`
   - Reply with confirmation: "Project **{name}** created! Category and channels ready. Use the thread above to describe your product."
   - Note: Does NOT call vco init/clone yet per D-01 (Phase 6 PM/Strategist drives that)

2. **`!dispatch [agent_id_or_all]`** per D-05, DISC-04:
   - Validate agent_manager is initialized
   - If argument is "all": `result = await asyncio.to_thread(self.bot.agent_manager.dispatch_all)`
   - Else: validate agent_id exists in config, `result = await asyncio.to_thread(self.bot.agent_manager.dispatch, agent_id)`
   - Reply with success/failure message

3. **`!status`** per D-04, DISC-05:
   - `status_text = await asyncio.to_thread(generate_project_status, self.bot.project_dir, self.bot.project_config)`
   - `embed = build_status_embed(status_text)`
   - `await ctx.send(embed=embed)`
   - Handle empty status gracefully (no agents dispatched yet)

4. **`!kill <agent_id>`** per D-03, D-05, DISC-07:
   - Requires confirmation per D-03: send message "Kill agent **{agent_id}**? This will terminate the session." with ConfirmView
   - Set `view.interaction_user_id = ctx.author.id`
   - `await view.wait()`
   - If view.value is True: `success = await asyncio.to_thread(self.bot.agent_manager.kill, agent_id)` then reply result
   - If view.value is False or None (timeout): reply "Kill cancelled."

5. **`!relaunch <agent_id>`** per D-05, DISC-08:
   - Validate agent_id exists in config
   - `result = await asyncio.to_thread(self.bot.agent_manager.relaunch, agent_id)`
   - Reply with success/failure

6. **`!standup`** per D-06, DISC-06:
   - Scaffold only: reply "Standup coming in Phase 7. Channel structure is ready."
   - This is a placeholder per D-06

7. **`!integrate`** per D-07, DISC-09:
   - Scaffold with confirmation per D-03: send "Trigger integration pipeline?" with ConfirmView
   - If confirmed: reply "Integration pipeline coming in Phase 7."
   - This is a placeholder per D-07

**Error handling:**
- Each command wrapped in try/except. On error, reply with "Error: {error}" using build_alert_embed.
- `cog_check(self, ctx)` method: return `self.bot.is_bot_ready` with "Bot is starting up" message if not ready (Pitfall 6).

**Module-level setup function:**
