**Create src/vcompany/daemon/runtime_api.py** with:

```python
"""RuntimeAPI -- typed gateway to CompanyRoot operations.

All methods are async. No discord.py imports. Uses CommunicationPort
for any outbound messaging. The daemon layer accesses CompanyRoot
exclusively through this class.
"""
```

class RuntimeAPI:
- Constructor takes:
  - `company_root: CompanyRoot`
  - `comm_port_getter: Callable[[], CommunicationPort]` (lazy getter -- CommunicationPort set after bot connects)
- Stores `self._root = company_root` and `self._get_comm = comm_port_getter`
- Stores `self._channel_ids: dict[str, str] = {}` (channel name -> channel ID registry)

Methods (all async):
- `hire(agent_id: str, template: str = "generic") -> str`: calls `self._root.hire(agent_id, template=template)` (NO guild param), returns `container.context.agent_id`. Before hire, call `self._get_comm().create_channel(CreateChannelPayload(category_name="vco-tasks", channel_name=f"task-{agent_id}"))`.
- `give_task(agent_id: str, task: str) -> None`: finds container via `self._root._find_container(agent_id)`, calls `container.give_task(task)`.
- `dismiss(agent_id: str) -> None`: calls `self._root.dismiss(agent_id)`.
- `status() -> dict`: returns `{"projects": {pid: {"agents": len(ps.children)} for pid, ps in self._root.projects.items()}, "company_agents": list(self._root._company_agents.keys())}`.
- `health_tree() -> dict`: calls `self._root.health_tree().model_dump()`.
- `register_channels(channels: dict[str, str]) -> None`: updates `self._channel_ids` with the dict. Non-async.
- `get_channel_id(name: str) -> str | None`: returns `self._channel_ids.get(name)`.

Import only from `vcompany.supervisor.company_root`, `vcompany.daemon.comm`, `vcompany.container.child_spec`, `vcompany.container.context`. NO discord.py imports.
