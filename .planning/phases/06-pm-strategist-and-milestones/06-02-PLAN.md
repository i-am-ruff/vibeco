       - `class StrategistConversation`:
         - `__init__(self, client: AsyncAnthropic, persona_path: Path | None = None, model: str = "claude-opus-4-6")`:
           - Load persona from file if exists, else use DEFAULT_PERSONA. Log warning if file missing.
           - `self._client = client`
           - `self._system_prompt: str` = loaded persona
           - `self._messages: list[dict] = []`
           - `self._total_tokens: int = 0`
           - `self._message_count_since_check: int = 0`
           - `self._lock = asyncio.Lock()`
           - `self._model = model`
         - `async def send(self, content: str, role: str = "user") -> AsyncGenerator[str, None]`:
           - Acquire self._lock
           - Append `{"role": role, "content": content}` to self._messages
           - Increment _message_count_since_check
           - Call `_maybe_check_tokens()` (rough estimate first, API call if needed)
           - If tokens over limit, call `_perform_knowledge_transfer()`
           - Stream response via `self._client.messages.stream(model=self._model, system=self._system_prompt, messages=self._messages, max_tokens=8192)`
           - Accumulate full text, yield chunks
           - Append `{"role": "assistant", "content": full_text}` to self._messages
           - Release lock
         - `async def _maybe_check_tokens(self) -> None`:
           - Rough estimate: sum of len(m["content"]) for all messages, divided by ROUGH_CHARS_PER_TOKEN
           - If rough estimate < TOKEN_CHECK_THRESHOLD: return (skip API call)
           - If _message_count_since_check < TOKEN_CHECK_INTERVAL and _total_tokens < TOKEN_CHECK_THRESHOLD: return
           - Call `self._client.messages.count_tokens(model=self._model, system=self._system_prompt, messages=self._messages)`
           - Update self._total_tokens
           - Reset _message_count_since_check = 0
         - `async def _perform_knowledge_transfer(self) -> None`:
           - Call `generate_knowledge_transfer(self._messages, self._system_prompt, self._total_tokens)`
           - Replace self._messages with `[{"role": "user", "content": f"[KNOWLEDGE TRANSFER]\n\n{kt_doc}\n\nPlease acknowledge you have received this knowledge transfer and are ready to continue."}]`
           - Reset self._total_tokens = 0
           - Log info about KT
         - `@property def messages(self) -> list[dict]`: return copy of self._messages
         - `@property def token_count(self) -> int`: return self._total_tokens
