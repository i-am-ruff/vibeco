Output: 5 working CLI commands registered in main.py, plus shared connection helper and tests.

Daemon socket methods already registered (daemon.py _register_socket_endpoints):
- "hire" -> params: {"agent_id": str, "template": str} -> returns: {"agent_id": str}
- "give_task" -> params: {"agent_id": str, "task": str} -> returns: {"status": "ok"}
- "dismiss" -> params: {"agent_id": str} -> returns: {"status": "ok"}
- "status" -> params: {} -> returns: {"projects": {pid: {"agents": N}}, "company_agents": [str]}
- "health_tree" -> params: {} -> returns: CompanyHealthTree dict (see RESEARCH.md Pattern 4)

Health tree dict structure:
```python
{
    "supervisor_id": "company-root",
    "state": "running",
    "projects": [{"supervisor_id": str, "state": str, "children": [{"report": {"agent_id": str, "state": str, "inner_state": str, "uptime": float, "error_count": int, "blocked_reason": str|None, "is_idle": bool}}]}],
    "company_agents": [{"report": {"agent_id": str, "state": str, ...}}]
}
```
