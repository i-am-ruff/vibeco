6. `async def get_agent_states(self) -> list[dict]` -- iterate all containers across company_agents and projects, return list of dicts with agent_id, agent_type, state fields.

7. `async def checkin(self) -> dict` -- gather checkin data. Import gather_checkin_data inside method, call it with self._root, return result dict.

8. `async def standup(self) -> dict` -- similar pattern, import StandupSession inside method.

9. `async def run_integration(self) -> dict` -- import IntegrationPipeline inside method, run it against project.

Each method should follow the existing pattern: find container via self._root._find_container(), raise KeyError on missing, use lazy imports for anything outside vcompany.daemon.
