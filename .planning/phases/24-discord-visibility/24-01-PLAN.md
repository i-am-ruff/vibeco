   - Imports: `logging`, `discord`, `discord.ext.commands`, TYPE_CHECKING guard for `VcoBot`, `AgentContainer`
   - Import `MessageContext` from `vcompany.models.messages`

   - Define `class MentionRouterCog(commands.Cog):` with docstring referencing D-01, D-02, D-04

   - `__init__(self, bot: VcoBot)`:
     - `self.bot = bot`
     - `self._agent_handles: dict[str, AgentContainer] = {}` -- maps handle string (e.g. "Strategist", "PMProjectX") to container
     - `self._handle_channels: dict[str, str] = {}` -- maps handle string to primary channel_id

   - `def register_agent(self, handle: str, container: AgentContainer, channel_id: str | None = None) -> None:`
     - Stores handle -> container mapping in `_agent_handles`
     - If channel_id provided, stores in `_handle_channels`
     - Logs: `"Registered agent handle @%s -> %s", handle, container.context.agent_id`

   - `def unregister_agent(self, handle: str) -> None:`
     - Removes from both dicts if present

   - `@commands.Cog.listener()` on `async def on_message(self, message: discord.Message) -> None:`
     - Skip if `message.author.bot` (D-04, prevents bot loop -- Pitfall 1)
     - Skip if `message.author.id == self.bot.user.id` (extra safety)
     - Scan `message.content` for `@{handle}` patterns for each registered handle
     - For each matching handle, call `await self._deliver_to_agent(handle, message)`

   - `async def _deliver_to_agent(self, handle: str, message: discord.Message) -> None:`
     - Build `MessageContext`:
       - `sender=message.author.display_name`
       - `channel=message.channel.name`
       - `content=message.content`
       - `message_id=str(message.id)`
     - If `message.reference` and `message.reference.message_id`:
       - Try `parent = await message.channel.fetch_message(message.reference.message_id)`
       - Set `context.parent_message = parent.content` and `context.is_reply = True`
       - Wrap in try/except `discord.NotFound` -- set `parent_message = None` on failure (D-15)
     - Call `await container.receive_discord_message(context)` where container = `self._agent_handles[handle]`
     - Wrap entire delivery in try/except Exception with logger.error (never crash the cog)

   - Add module-level `async def setup(bot: VcoBot) -> None:` that creates and adds the cog
