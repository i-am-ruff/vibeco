   ```python

       self,
       memory_store: MemoryStore,
       on_mutation: Callable[[str], Awaitable[None]] | None = None,
   ):
   ```
   - Add `from typing import Awaitable, Callable` to imports (keep existing imports)
   - Store as `self._on_mutation = on_mutation`

2. Add a private helper method:
   ```python
   async def _notify(self, message: str) -> None:
       """Fire mutation callback if registered. Never raises."""
       if self._on_mutation is not None:
           try:
               await self._on_mutation(message)
           except Exception:
               logger.warning("Backlog mutation callback failed for: %s", message, exc_info=True)
   ```

3. Add `await self._notify(...)` call at the END of each mutation method (post-commit per D-06 recommendation, after `_persist()`):

   - `append()`: After persist, add:
     `await self._notify(f"[Backlog] Added: '{item.title}' (priority {item.priority})")`

   - `insert_urgent()`: After persist, add:
     `await self._notify(f"[Backlog] Urgent: '{item.title}' inserted at position 0")`

   - `reorder()`: After persist, add:
     `await self._notify(f"[Backlog] Reordered: '{item_id}' moved to position {new_position}")`

   - `cancel()`: After persist, add:
     `await self._notify(f"[Backlog] Cancelled: '{item_id}'")`

   - `claim_next()`: After persist (only if an item was claimed, inside the `if` block), add:

     `await self._notify(f"[Backlog] Completed: '{item_id}'")`

   - `mark_pending()`: After persist, add:
