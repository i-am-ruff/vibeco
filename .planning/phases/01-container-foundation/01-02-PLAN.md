    Create `src/vcompany/container/memory_store.py` with class MemoryStore:

    ```python
    class MemoryStore:
        def __init__(self, db_path: Path):
            self._db_path = db_path
            self._db: aiosqlite.Connection | None = None
    ```

    Methods:
    - `async def open(self) -> None`: mkdir parents, connect via aiosqlite, enable WAL mode (`PRAGMA journal_mode=WAL`), create tables:
      - `kv` table: key TEXT PRIMARY KEY, value TEXT NOT NULL, updated_at TEXT NOT NULL
      - `checkpoints` table: id INTEGER PRIMARY KEY AUTOINCREMENT, label TEXT NOT NULL, data TEXT NOT NULL, created_at TEXT NOT NULL
    - `async def close(self) -> None`: close connection if open, set self._db = None
    - `async def get(self, key: str) -> str | None`: SELECT value FROM kv WHERE key = ?
    - `async def set(self, key: str, value: str) -> None`: INSERT OR REPLACE with datetime.now(timezone.utc).isoformat()
    - `async def delete(self, key: str) -> None`: DELETE FROM kv WHERE key = ?
    - `async def keys(self) -> list[str]`: SELECT key FROM kv ORDER BY key
    - `async def checkpoint(self, label: str, data: str) -> None`: INSERT into checkpoints with timestamp
    - `async def get_latest_checkpoint(self, label: str) -> str | None`: SELECT data FROM checkpoints WHERE label = ? ORDER BY id DESC LIMIT 1
    - `async def list_checkpoints(self) -> list[dict[str, str]]`: SELECT label, created_at FROM checkpoints ORDER BY id DESC — returns list of {"label": ..., "created_at": ...}

    All datetime values use `datetime.now(timezone.utc).isoformat()`.
