    **Create `src/vcompany/cli/clone_cmd.py`:**

    ```python
    import click
    import shutil
    from pathlib import Path
    from vcompany.models.config import load_config, AgentConfig, ProjectConfig
    from vcompany.git import ops as git
    from vcompany.shared.file_ops import write_atomic
    from vcompany.shared.templates import render_template

    PROJECTS_BASE = Path.home() / "vcompany" / "projects"

    @click.command()
    @click.argument("project_name")
    @click.option("--force", is_flag=True, default=False,
                  help="Delete and re-clone existing agent directories")
    def clone(project_name: str, force: bool):
        """Clone repos and deploy artifacts for all agents in a project."""
    ```

    Implementation steps inside `clone`:

    1. Resolve project dir: `PROJECTS_BASE / project_name`. Verify it exists (must run `vco init` first).
    2. Load config: `load_config(project_dir / "agents.yaml")`. Fail with clear error if invalid.
    3. For each agent in `config.agents`:
       a. Set `clone_dir = project_dir / "clones" / agent.id`
       b. If clone_dir exists and `--force`: delete it via `shutil.rmtree`. If exists and not force: skip with warning "Clone already exists for {agent.id}, use --force to re-clone".
       c. Run `git.clone(config.repo, clone_dir)`. On failure, print error and continue to next agent (don't leave partial state -- if clone_dir was created but git failed, rmtree it).
       d. Create agent branch: `git.checkout_new_branch(f"agent/{agent.id.lower()}", cwd=clone_dir)` (per Pitfall 7: lowercase, hyphen-separated).
       e. Deploy all artifacts to the clone using `_deploy_artifacts(clone_dir, agent, config)`.

    4. Print summary: "Cloned {n} agents for project '{project_name}'"

    **`_deploy_artifacts(clone_dir, agent, config)` function:**

    Deploy all 5 artifact types per D-07:
