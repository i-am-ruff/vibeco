    2. `@dataclass AgentWorkflowState`:
       - agent_id: str
       - current_phase: int = 0
       - stage: WorkflowStage = WorkflowStage.IDLE
       - stage_started_at: float = 0.0
       - blocked_since: float | None = None
       - blocked_reason: str = ""

    3. Module-level `STAGE_COMPLETE_PATTERNS` dict mapping stage names to compiled regexes:
       - "discuss": `re.compile(r"discuss-phase complete", re.IGNORECASE)`
       - "plan": `re.compile(r"plan-phase complete", re.IGNORECASE)`
       - "execute": `re.compile(r"execute-phase complete", re.IGNORECASE)`
       - "research": `re.compile(r"research-phase complete", re.IGNORECASE)`

    4. Module-level `AGENT_ID_PATTERN = re.compile(r"^\S+\s+(\S+):\s+(.+)$")` to extract agent_id from vco report format "{timestamp} {agent_id}: {status_text}"

    5. `detect_stage_signal(message_content: str) -> tuple[str, str] | None`:
       - Parse agent_id from message using AGENT_ID_PATTERN
       - Check status text against STAGE_COMPLETE_PATTERNS
       - Return (agent_id, stage_name) or None

    6. `class WorkflowOrchestrator`:
       - `__init__(self, project_dir: Path, config: ProjectConfig, agent_manager: AgentManager)`:
         - Store references. Initialize `_agent_states: dict[str, AgentWorkflowState] = {}`
       - `start_agent(self, agent_id: str, phase: int) -> bool`:
         - Create AgentWorkflowState with stage=DISCUSS, current_phase=phase
         - Call `self._agent_manager.send_work_command(agent_id, f"/gsd:discuss-phase {phase}", wait_for_ready=True)`
         - Set stage_started_at to time.monotonic()
         - Return send result
       - `on_stage_complete(self, agent_id: str, stage: str) -> WorkflowStage`:
         - Transition logic: discuss->DISCUSSION_GATE, plan->PM_PLAN_REVIEW_GATE, execute->PHASE_COMPLETE
         - Update stage_started_at, clear blocked state
         - Return new stage
       - `advance_from_gate(self, agent_id: str, approved: bool) -> bool`:
         - If at DISCUSSION_GATE and approved: send `/gsd:plan-phase {phase}`, transition to PLAN
         - If at DISCUSSION_GATE and rejected: re-send `/gsd:discuss-phase {phase}`, stay DISCUSS
         - If at PM_PLAN_REVIEW_GATE and approved: send `/gsd:execute-phase {phase}`, transition to EXECUTE
         - If at PM_PLAN_REVIEW_GATE and rejected: re-send `/gsd:plan-phase {phase}`, stay PLAN
         - Return send_work_command result
       - `get_agent_state(self, agent_id: str) -> AgentWorkflowState | None`
       - `get_all_states(self) -> dict[str, AgentWorkflowState]`
       - `recover_from_state(self, agent_id: str, clone_dir: Path) -> WorkflowStage`:
         - Read clone_dir / ".planning" / "STATE.md"
         - Parse `stopped_at` and `status` fields (simple string matching)
         - Map to appropriate stage: "context gathered" -> DISCUSSION_GATE, "planned" -> PM_PLAN_REVIEW_GATE, "executing" -> EXECUTE, etc.
         - Set state and return stage
       - `handle_unknown_prompt(self, agent_id: str, prompt_text: str) -> str`:
         - Set blocked_since = time.monotonic(), blocked_reason = prompt_text[:200]
         - Return f"Agent {agent_id} blocked on unknown prompt: {prompt_text[:100]}"
       - `check_blocked_agents(self, timeout: float = 600.0) -> list[AgentWorkflowState]`:
         - Return agents where blocked_since is not None and (now - blocked_since) > timeout
